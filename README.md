# Multi-Office-VLAN-Wireless-Network-Design
“This project demonstrates a network design for two offices, Melbourne and Sydney, with VLANs for IT, Finance, and Accounting departments. A separate wireless network is provided for guest users. The design includes DHCP configuration, router sub-interfaces for VLAN routing, and WAN connectivity using public IPs.”
